# Fonts

Self-hosted woff2 files for Space Grotesk, Inter and JetBrains Mono
(DESIGN.md §2.3) are not sourced yet. `tokens.css` references the family
names with system-font fallbacks so the stack renders correctly today.
Real `.woff2` binaries and the corresponding `@font-face` rules land in M5
per the design doc's phased plan (§10).
